Étape de création d'un nouveau magasin en ligne

 1. Connectez vous à votre cPanel

   2. Cliquez sur l'icone de WordPress.

   3. Cliquez sur le boutton d'installation

   4. Entrez les informations suivantes comme:

       a. Sélectionner l'address du domain voulu
       b. Le nom de votre site web
       c. Entrez le mot de passe que vous voulez
       d. Entrez votre adresse courriel

   5. Cliquez sur l'onglet Plug-in

   6. Dans la barre de recherche, chercher "WooCommerce" et installer le

  Votre site web est installer et prêt a être personnaliser 
  Il ne reste plus qu'a vous connecter via votre navigateur à l'url (nom de domaine) que vous aviez choisit.
